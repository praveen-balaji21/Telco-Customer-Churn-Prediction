AI-Powered Customer Churn Prediction

Predict customer churn using machine learning techniques and identify the key factors that influence customer retention.

📌 Project Overview

Customer churn is one of the biggest challenges faced by subscription-based businesses. This project uses historical customer data from a telecommunications company to predict whether a customer is likely to leave the service.

The model analyzes customer demographics, account information, service subscriptions, and billing details to identify customers at risk of churning.

🎯 Objectives
Perform data cleaning and preprocessing
Conduct exploratory data analysis (EDA)
Build and train a machine learning model for churn prediction
Evaluate model performance using classification metrics
Identify important features that contribute to customer churn
🗂️ Dataset

This project uses the IBM Telco Customer Churn Dataset.

Dataset Source:

 Telco Customer Churn Dataset

Dataset Features
Customer Information: Gender, Senior Citizen, Partner, Dependents
Account Information: Tenure, Contract Type, Payment Method
Services: Internet Service, Phone Service, Streaming Services
Billing Information: Monthly Charges, Total Charges
Target Variable: Churn (Yes/No)
